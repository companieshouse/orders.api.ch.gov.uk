#Companies House Orders API
##orders.api.ch.gov.uk
CH API handling CRUD operations on CH Ordering Service

###Requirements
* Java 8
* Maven
* Git

###Getting Started
1. Run `make` to build
2. Run `./start.sh` to run

###Endpoints
| Page | URL | Response
--- | --- | ---
*Health check* | *`/healthcheck`* | *`200`*
